# MS_AYAX_BANCOLOMBIA


### How to compile

- Requirements
    - NodeJS version >= 4
    - NPM Global Dependencies:
        - better-npm-run
        - webpack

> npm install -g better-npm-run webpack
        
- Commands:
    1. Go to the project folder
    2. Run ```npm run build```
    3. Wait and then go to the build folder created by webpack.
